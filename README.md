
# Deathrun Game - Frontend

This repository contains the frontend code for the Deathrun Game, built using Vite, React, and TypeScript.

## Installation

To install and run the frontend locally, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/CodingFactory-Repos/deathrun-frontend.git
   ```

2. Navigate to the project directory:
   ```
   cd deathrun-frontend
   ```

3. Install the dependencies using pnpm:
   ```
   pnpm install
   ```

4. Start the development server:
   ```
   pnpm run dev
   ```

5. The application will run on `http://localhost:3000`.

## Project Overview

The frontend allows the trap setter to visualize and interact with the game room, placing traps and adjusting their strategy in real-time.

### Key Features:
- **Room Visualization:** View the room from a top-down perspective.
- **Trap Placement:** Choose and place traps on the interface before the player starts their run.
- **Score Tracking:** Monitor the player’s performance and adjust trap placement accordingly.
